# Computer-Builder
## 概要
ユーザーはCPU、GPU、メモリーカード（RAM）（1 - 4 メモリモジュール）、ストレージ（HDD または SSD）を指定して、、コンピューターを作成するウェブアプリケーションです。</br>
ユーザーはブランド、個数、ストレージを指定でき、このアプリケーションはそれらの情報をもとに、エンドポイントから条件に合うモデルを探します。</br>
全てのパーツを選んだ後、「Add PC」を押すと全ての情報を考慮した仕事用及びゲーム用の各ベンチマークを表示します。</br>

## URL
このURLからアプリケーションを見ることができます。</br>
